# 话图能画什么（v0.12.1）

## 支持
1. **平面几何**：三角形（等腰/等边/直角）、圆（切线/弦/直径/内接四边形/内切外接圆）、四边形（正方形/矩形/菱形/平行四边形/梯形）
2. **常见辅助线**：中线、高、角平分线、垂直平分线、垂足、中点
3. **坐标系**：axis + 网格 + 刻度 + 箭头
4. **几何变换**：旋转、平移、反射（轴对称）、中心对称 —— 生成虚线派生图形
5. **函数图像**：显式 y=f(x) 或 x=g(y)，支持 sin/cos/tan/sqrt/exp/log/abs/pow 和常量 pi/e
6. **抛物线 y²=2x**：拆成两条曲线 y=±√(2x) 显式表达
7. **on_curve 硬约束**：点严格在函数曲线上
8. **半平面约束**：same_side / opposite_side（"C 在 AB 上方"）

## 不支持
1. **立体几何**：棱锥、棱柱、圆柱、圆锥、球、三视图、展开图
2. **椭圆/双曲线一般式**：x²/a²+y²/b²=1（隐式方程），除非老师明确要求拆解为两条 y=... 显式函数
3. **统计图表**：柱状图、饼图、折线图、直方图、条形图、扇形图
4. **具体坐标值描述**：如 "A(2, 3)"、"B(-1, 5)" 类硬编码点位（可以画坐标系但不接受具体点坐标）
5. **数轴上的点位关系**（较边缘，会 refuse）
6. **求代数最小值/最大值/证明**：这不是作图任务

# 任务

我给你一段来自中考/高考数学试卷的**题目原文**。请判断：

1. 这道题**是否是作图题**（含"如图"、"画"、"作"、几何关系、函数图像等描述性绘图要求）
2. 若是作图题：
   - 提取**一句作图指令**（简洁中文，去掉证明/求值/求最小值部分，只保留"画什么"）
   - 标注**题型分类**：`平面几何` / `坐标系` / `函数图像` / `几何变换` / `立体几何` / `统计图表` / `数轴` / `其它`
   - 判断**话图预期**：`ok`（能画）/ `partial`（能画一部分但有超出能力的要求）/ `refuse`（属于不支持类型）
   - 给出**置信度**：`high` / `medium` / `low`
3. 若不是作图题（纯代数、纯计算、选择题选项、非几何证明等），返回 `is_drawing_task: false`

# 输出格式

严格输出 JSON，禁止 Markdown 代码块、禁止解释性文字，外层只有一个 JSON 对象。

若是作图题：
```
{"is_drawing_task": true, "nl": "...", "type": "...", "expected": "ok|partial|refuse", "confidence": "high|medium|low", "notes": "简短说明（可选）"}
```

若不是作图题：
```
{"is_drawing_task": false, "reason": "简短说明为什么不算作图题"}
```

# 示例

## 示例 1（是作图题，ok）
输入：
> 5．（3 分）如图，在△ABC 中，AB=AC，D 是 BC 的中点，DE⊥AB 于 E，DF⊥AC 于 F，求证：DE=DF。

输出：
{"is_drawing_task": true, "nl": "画等腰三角形 ABC，AB=AC，D 是 BC 的中点，过 D 作 DE⊥AB 于 E，作 DF⊥AC 于 F", "type": "平面几何", "expected": "ok", "confidence": "high", "notes": "求证部分不入 DSL，只画图"}

## 示例 2（是作图题，refuse - 立体）
输入：
> 12．画四棱锥 P-ABCD，底面 ABCD 为正方形，PA⊥底面 ABCD，PA=AB=2。

输出：
{"is_drawing_task": true, "nl": "画四棱锥 P-ABCD，底面正方形，PA⊥底面，PA=AB=2", "type": "立体几何", "expected": "refuse", "confidence": "high", "notes": "立体几何不在话图 v0.12.1 能力范围"}

## 示例 3（是作图题，partial - 函数交点）
输入：
> 画反比例函数 y=6/x 与直线 y=x-1 的图像，标出交点。

输出：
{"is_drawing_task": true, "nl": "画反比例函数 y=6/x 与直线 y=x-1 的图像，标出它们的交点", "type": "函数图像", "expected": "partial", "confidence": "medium", "notes": "两条曲线能画，但'交点'需要求解，LLM 可能只画曲线不标交点"}

## 示例 4（不是作图题）
输入：
> 计算：√12 - 3√(1/3) = ?

输出：
{"is_drawing_task": false, "reason": "纯代数计算题，无几何图形"}

## 示例 5（不是作图题 - 选择题选项）
输入：
> A．圆柱          B．圆锥          C．三棱柱          D．球

输出：
{"is_drawing_task": false, "reason": "选择题的选项列表，不是完整题目"}

# 特别提示

- 若原文以"下列..."、"若..."、"已知..."开头但不涉及作图 → `is_drawing_task: false`
- 若原文含"如图"但你无法从文字判断图形（因为原图是图片） → 尝试从条件反推，若能推 → is_drawing_task=true；若完全无法推 → is_drawing_task=false, reason 写"依赖题图，纯文本无法重建"
- **nl 一句话说清作图任务**，不要写多句、不要写"求证"、"求值"、"解答"等
- 数轴题（如"在数轴上表示 √2"）标为 `type: 数轴, expected: refuse`
- 概率/统计/数据分析 → `is_drawing_task: false`（除非明确画柱状图 → refuse）
- 若一道大题分多个小问且其中一问是作图，只提取作图那一问
- **不要生成 DSL**，只生成一句作图指令 `nl`
