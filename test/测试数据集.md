# talk2graph 评估测试集

> 用于评估 LLM 生成 TikZ 数学图形的质量。
> 每个用例包含：自然语言描述、难度等级、评估维度、检查要点。

## 评估维度

每个测试用例从四个维度评估：

| 维度 | 说明 | 权重 |
|---|---|---|
| 几何正确性 | 长度比例、角度、形状是否符合要求 | 40% |
| 标注完整性 | 顶点标注、边长标注、角度标注是否齐全 | 25% |
| 视觉清晰度 | 线宽、颜色、位置是否清晰可读 | 20% |
| 中文标注 | 标注文字是否正确使用中文 | 15% |

评分：0-5 分，3 分及格，4 分良好，5 分优秀。

## 难度等级

- ⭐ 基础：单一简单图形，K12 教材常见
- ⭐⭐ 中等：带标注、多元素组合
- ⭐⭐⭐ 困难：复杂标注、多图形叠加、函数图像

## 使用方法

### 自动评估流程

```
for each test case:
    1. 发送 natural_language_prompt → LLM → 得到 TikZ 源码
    2. TikZ 源码 → 渲染器 → SVG
    3. 人工评分：几何正确性(0-5) / 标注完整性(0-5) / 视觉清晰度(0-5) / 中文标注(0-5)
    4. 记录总分和失败原因
```
