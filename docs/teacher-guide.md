# 话图 T2G — 老师内测说明

欢迎试用「话图 T2G」！这是一个**用一句话画初中几何图形**的工具。

## 怎么打开

浏览器访问：**http://t2g.example.com**（部署后填正式地址）

> 推荐 Chrome / Edge / 火狐最新版；目前不支持 IE 和老版本 Safari。

## 第一次用，跟着做 3 件事

### 1. 让 AI 画一张图

左侧对话框输入：

```
画一个内切圆半径为 3 的等腰三角形
```

按 **⌘+Enter / Ctrl+Enter** 发送。你的话会**立即**出现在对话区，下面是「话图正在思考中…」的占位提示，约 5-10 秒后画板就会出现等腰三角形和它的内切圆。

### 2. 继续修改

试试发：

```
把内切圆半径改成 5
```

或

```
在底边上加一个中点 M
```

AI 会基于刚才的图继续改。

### 3. 精细调整

- 右上角"对象"列表 → 点任意约束的数字 → 直接改
- 鼠标拖动图上的点 → 重新摆位（约束仍然满足）
- 顶栏「撤销 / 重做」可以来回切换

### 4. 给画板反馈（重要！）

每次画完图，画板**右下角**会出现两个按钮：

- 👍 **不错** — 一键告诉我们这张画对了
- 👎 **不对** — 点击后会弹出一个输入框，可以写一句"哪里不对"

你的反馈会进我们的数据库，帮助我们改 prompt 和 DSL。

### 5. 导出到课件

顶栏「导出 ▾」选 SVG / PNG / PDF。
- **PPT**：粘贴 SVG 进 PPT 2019 / WPS 即可，矢量不糊
- **试卷**：选 PDF 最稳

## 你可以试这些题型

### ✅ 支持的题型（MVP 范围）

| 类别 | 例句 |
|---|---|
| 三角形 | 画一个等边三角形 边长 6 |
| 三角形 + 圆 | 画直角三角形 ABC，C 是直角，BC=3 CA=4，再画外接圆 |
| 中线/高/角平分线 | 三角形 ABC 中，AD 是角 A 的平分线，D 在 BC 上 |
| 四边形 | 画一个菱形 ABCD，边长 4，∠A=60 |
| 圆 | 圆 O 半径 5，A、B 在圆上，∠AOB = 90° |
| 圆 + 切线 | 圆 O 半径 5，T 在圆上，AT 切圆 O 于 T，AT=12 |
| 四点共圆 | 画一个四边形 ABCD 内接于圆 |

### ⚠️ 暂不支持的题型（V2 计划）

| 类别 | 触发的提示 |
|---|---|
| 函数图像（y=x²、y=sin x 等）| 🟡「话图当前版本暂不支持函数图像和圆锥曲线，这一类计划在 V2 中支持」 |
| 圆锥曲线（抛物线、椭圆、双曲线）| 🟡 同上 |
| 立体几何（棱锥、圆柱、三视图）| 🟡「话图当前版本只支持平面几何…」 |
| 统计图表（柱状图、饼图、直方图）| 🟡「话图当前版本不支持统计图表…」 |
| 直接给坐标（A(2,3)）| 🟡「请改用边长、角度等几何关系描述」 |

如果遇到上面的黄色提示**不是 bug**，是 AI 主动识别为不支持的题型。你可以试着改用平面几何描述。

## 看到不同颜色的消息是什么意思

| 颜色 | 含义 |
|---|---|
| 🟡 黄色（拒绝） | AI 识别为不支持的题型，并给出建议 |
| 🟣 紫色（约束失败）| 题目里的几何条件互相矛盾，求解器无法满足 |
| 🔴 红色（系统错误）| 网络问题或 LLM 服务不可用，请稍后重试或切换 Provider |

## 数据与隐私

- 你提交的题目、生成的图、👍/👎 反馈都会保存在我们服务器上，便于下次打开继续编辑、并帮助我们改进产品
- 服务器在腾讯云国内节点；老师题目**不会**用于训练任何模型
- 想删除某个会话：点该会话 → 删除（清服务器 + 本地）
- 想全部清除：浏览器清 LocalStorage + 联系我们删后端记录

## 已知限制（V1）

- 暂不支持立体几何 / 函数图像 / 圆锥曲线 / 统计图表（V2 计划）
- 暂不支持直接坐标描述（V2 会加坐标系支持）
- 复杂题（超过 10 个对象 / 多重相似）可能多解，AI 会挑一个，你可以拖动调整
- 中文标签在 PPT 里可能受字体影响；试卷场景建议用 PDF
- 部分推理模型（如 DeepSeek v4-pro）延迟较高，建议默认使用火山 GLM-5.2 或 DeepSeek v4-flash
