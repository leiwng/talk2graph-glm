# 话图 T2G — 老师内测说明

欢迎试用「话图 T2G」！这是一个**用一句话画初中数学图形**的工具。

> **当前版本 v0.12.1（V2 主线 + W12 LLM 行为优化）**：在 V1 平面几何基础上，新增坐标系、几何变换、函数图像、on_curve 硬约束四大能力。

## 怎么打开

浏览器访问：**http://49.233.15.73:8080**

> 推荐 Chrome / Edge / 火狐最新版；目前不支持 IE 和老版本 Safari。
> **首次打开或版本更新后请 Ctrl+F5 强制刷新**，清掉浏览器缓存里的旧界面。

## 第一次用，跟着做 3 件事

### 1. 让 AI 画一张图

左侧对话框输入：

```
画一个内切圆半径为 3 的等腰三角形
```

按 **⌘+Enter / Ctrl+Enter** 发送。你的话会**立即**出现在对话区，下面是「话图正在思考中…」的占位提示，约 5-10 秒后画板就会出现等腰三角形和它的内切圆。

### 2. 继续修改

试试发：

```
把内切圆半径改成 5
```

或

```
在底边上加一个中点 M
```

AI 会基于刚才的图继续改。

### 3. 精细调整

- 右上角"对象"列表 → 点任意约束的数字 → 直接改
- 鼠标拖动图上的点 → 重新摆位（约束仍然满足）
- 顶栏「撤销 / 重做」可以来回切换

### 4. 给画板反馈（重要！）

每次画完图，画板**右下角**会出现两个按钮：

- 👍 **不错** — 一键告诉我们这张画对了
- 👎 **不对** — 点击后会弹出一个输入框，可以写一句"哪里不对"

你的反馈会进我们的数据库，帮助我们改 prompt 和 DSL。

### 5. 导出到课件

顶栏「导出 ▾」选 SVG / PNG / PDF。
- **PPT**：粘贴 SVG 进 PPT 2019 / WPS 即可，矢量不糊
- **试卷**：选 PDF 最稳

## 你可以试这些题型

### ✅ V1 · 平面几何

| 类别 | 例句 |
|---|---|
| 三角形 | 画一个等边三角形 边长 6 |
| 三角形 + 圆 | 画直角三角形 ABC，C 是直角，BC=3 CA=4，再画外接圆 |
| 中线/高/角平分线 | 三角形 ABC 中，AD 是角 A 的平分线，D 在 BC 上 |
| 四边形 | 画一个菱形 ABCD，边长 4，∠A=60 |
| 圆 | 圆 O 半径 5，A、B 在圆上，∠AOB = 90° |
| 圆 + 切线 | 圆 O 半径 5，T 在圆上，AT 切圆 O 于 T，AT=12 |
| 四点共圆 | 画一个四边形 ABCD 内接于圆 |

### ✅ V2-A · 坐标系 (v0.9.0 新增)

| 例句 | 说明 |
|---|---|
| 画一个平面直角坐标系 | 默认 x/y ∈ [-5, 5]、刻度 1 |
| 画一个坐标系，x 轴 -10 到 10，y 轴 -6 到 6，刻度为 2 | 自定义范围与刻度 |
| 在方格纸上画正方形 ABCD 和三角形 EFG，顶点都在格点上 | 坐标系 + 图形组合 |

**方位描述**（v0.10.0 新增）：

| 例句 | 说明 |
|---|---|
| 画直角三角形 ABC，C 为直角顶点，C 在 AB 上方，BC=3 CA=4 | 稳定输出 C 在指定方位 |

### ✅ V2-W11 · 几何变换 (v0.11.0 新增)

| 类型 | 例句 |
|---|---|
| 中心对称 | 画三角形 ABC，AB=4 BC=3 ∠B=60°；作三角形 ABC 关于点 B 的中心对称图形 A'B'C' |
| 旋转 | 画直角三角形 ABC，∠ABC=90° AB=1 BC=2；将线段 AC 绕点 A 旋转 90° 得到线段 AD |
| 平移 | 画三角形 ABC；沿 AB 方向平移 3 单位得到 A'B'C' |
| 轴对称 | 画三角形 ABC；关于直线 MN 对称得到 A'B'C' |

派生图形会用**虚线**画出，与原图区分；派生顶点自动加撇（A → A'）。

### ✅ V2-B · 函数图像 (v0.12.0 新增) + W12 (v0.12.1)

**前提**：函数图像必须画在坐标系里。若一句话里只提函数、没提坐标系，AI 会自动加。

| 类型 | 例句 |
|---|---|
| 一次函数 | 画 y = 2x + 1 的图像 |
| 二次函数 | 画 y = x² 的图像，x 从 -3 到 3 |
| 反比例函数 | 画 y = 1/x 的图像 |
| 三角函数 | 画 y = sin(x) 的图像 |
| 指对数 | 画 y = log(x) 的图像 |
| 抛物线（含隐式） | 画抛物线 y²=2x 及其准线 |
| 点在曲线上（W12） | 在 y=6/x 上取 A、B 两点，直线 AB 过原点 |

**表达式语法**（Python 风格）：
- 幂：`x**2` 或 `pow(x, 2)`
- 开根：`sqrt(x)`
- 三角：`sin(x)` / `cos(x)` / `tan(x)`
- 对数指数：`log(x)` / `log10(x)` / `exp(x)`
- 常量：`pi` / `e`

### ⚠️ 暂不支持的题型

| 类别 | 触发的提示 |
|---|---|
| 立体几何（棱锥、圆柱、三视图）| 🟡「话图当前版本只支持平面几何，立体几何计划在 V3 中支持」 |
| 统计图表（柱状图、饼图、直方图）| 🟡「话图当前版本不支持统计图表」 |
| 直接给具体坐标值 A(2,3) | 🟡「请改用边长、角度等几何关系描述，或先画一个坐标系再独立描述图形」 |
| 椭圆/双曲线的隐式一般式 x²/a²+y²/b²=1 | 🟡「暂不支持椭圆/双曲线一般式，可拆成 y=±b·√(...) 显式函数」 |

如果遇到上面的黄色提示**不是 bug**，是 AI 主动识别为不支持的题型。可以按照建议改用其他方式描述。

## 看到不同颜色的消息是什么意思

| 颜色 | 含义 |
|---|---|
| 🟡 黄色（拒绝） | AI 识别为不支持的题型，并给出建议 |
| 🟣 紫色（约束失败）| 题目里的几何条件互相矛盾，求解器无法满足 |
| 🔴 红色（系统错误）| 网络问题或 LLM 服务不可用，请稍后重试或切换 Provider |
| 灰色小字（重新理解）| AI 第一次输出与现有图形冲突，自动重画了，不用管 |

## 数据与隐私

- 你提交的题目、生成的图、👍/👎 反馈都会保存在我们服务器上，便于下次打开继续编辑、并帮助我们改进产品
- 服务器在腾讯云国内节点；老师题目**不会**用于训练任何模型
- 想删除某个会话：点该会话 → 删除（清服务器 + 本地）
- 想全部清除：浏览器清 LocalStorage + 联系我们删后端记录

## 已知限制（v0.12.1）

- 复杂题（超过 15 个对象 / 多重相似 / 长约束链）可能多解，AI 会挑一个，你可以拖动调整
- 中文标签在 PPT 里可能受字体影响；试卷场景建议用 PDF（V2-C 计划做 outline 化解决）
- 每次调用 LLM 5-15 秒；SSE 流式在后续版本上线
- 立体几何 / 统计图表 / 椭圆隐式方程 计划 V3 支持
